## Pui Pui System 期末個人專案系統
### 說明:

PuiPuiSystem資料夾內的檔案全為本專案的檔案

### 如何使用:
1. 請先安裝Xampp
2. 刪除Xampp安裝根目錄底下的htdocs資料夾(全部刪除)
3. 複製PuiPuiSystem資料夾裡的htdocs資料夾到XAMPP安裝根目錄
4. 開啟Xampp後啟動Apache與MySQL
5. 進入http://localhost/phpmyadmin/
6. 匯入CreateDB.sql
